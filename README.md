# Calculadora


1 - Executar as quatro operações aritméticas básicas: adição, subtração, multiplicação e divisão

2 - Deve ter seu código escrito completamente por você, sem o uso de bibliotecas, especialmente javascript

3 - Exceção feita ao estilo da página, que pode ser feito utilizando bootstrap
Não existe um padrão para estilo da sua página, mas o estilo comporá parte da nota.

4 - Você deve ter pelo menos uma página HTML, com um selectbox, dois inputs e um button.
5 - O selectbox terá as opções das operações matemáticas. Ao selecionar uma delas, por exemplo, adição, sua página deverá:
Escrever Adição em um título acima dos inputs
Escrever o sinal + entre os dois inputs
Quando o botão for pressionado, deve calcular o resultado e mostrar na tela.
Para o caso da divisão, caso o dividendo seja 0, você deve imprimir uma mensagem de erro: "Não é possível dividir por zero" (e não deve fazer o cálculo).
Sua entrega deve ser feita através de um repositório GIT, com sua página publicada no github pages

A atividade poderá ser entregue em duplas. Neste caso é importante informar os elementos da equipe.
Rubrica de avaliação:
Construção da página (HTML): 15%
Estilo (CSS): 15%
Manipulação do selectbox: 20%
Operações matemáticas: 30%
Tratamento do erro da divisão: 20%
